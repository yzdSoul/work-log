# 工作日志 — 2026-05-22

## 1. WeChat 群聊集成探索

### 目标
用户希望通过 OpenClaw/WeChat 桥接将 Hermes Agent 拉入微信群聊，并测试群聊功能。

### 已完成
1. **分析群聊接入方案** — 讨论了三种 WeChat 接入方式：
   - 个人微信桥接（OpenClaw/wechaty）
   - 企业微信群机器人（WeCom）
   - 微信公众号
2. **解码群二维码** — 用户发送微信群二维码图片，使用 pyzbar + libzbar0 + Pillow 成功解码，获得群邀请 URL
3. **验证邀请链接** — 通过浏览器访问 WeChat 群邀请 URL，确认跳转到下载页面（微信仅限移动端扫码）
4. **确认 pupppet 限制** — 当前 OpenClaw WeChat 桥接的 puppet 不支持群聊邀请功能
5. **给出替代方案** — 推荐 Telegram、飞书、Discord 作为群聊替代平台
6. **更新技能库** — 新增 `hermes-agent/references/weixin-setup.md` 参考文档，记录 WeChat 集成限制和二维码解码方法

### 待办
- [ ] 如需要群聊功能，切换到 Telegram / 飞书 / Discord 通道
- [ ] 测试 OpenClaw puppet 更新后是否支持群聊
